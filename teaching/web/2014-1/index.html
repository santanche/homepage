Web and Semantic Web (2014.1)

MO826/MC936 - Information Systems Topics

This discipline focuses on the subjects: Web and Semantic Web. Web subject: Web science and Web engineering; architecture, models, standards and languages; querying; social Web and social networks; content, behavioral and graph analysis; mining, search and deep Web; Web platform and applications; Web services; Web for mobiles; Web of Things; information extraction, matching and crowdsourcing. Semantic Web subject: architecture, models, standards and languages; Web of data, metadata, querying, rules and reasoning; data spaces, data integration and Linked Data; knowledge representation, formalisms and ontologies, semantic Web services.
