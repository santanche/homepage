Instalando e testando o Eclipse Modeling

O Eclipse é um ambiente integrado de desenvolvimento, que não apenas oferece ferramentas para o desenvolvimento e manutenção de uma grande diversidade de artefatos computacionais, como também dá suporte a tarefas de versionamento e colaboração. Por esta razão ele foi eleito não apenas para desenvolvimento de código Java, como também para outras tarefas relacionadas ao curso, tal como a modelagem.

Para se customizar o Eclipse para as diversas tarefas são usados plugins. Atualmente a variedade de plugins é tal, que foram criadas versões do Eclipse para download (pacotes) agrupando plugins tematicamente. Há pacotes para linguagens (Java, JavaScript, C etc.), como também para domínios específicos de desenvolvimento de aplicações (serviços Web, relatórios, modelagem etc.). O pacote de modelagem inclui também suporte completo para desenvolvimento e Java, por esta razão ele foi escolhido para ser usado como base no curso. A seguir seu tutorial de instalação:
